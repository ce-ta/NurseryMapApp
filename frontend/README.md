# NurseryMapApp

## 本システムの趣旨
地図上で保育園の位置を可視化し、各施設の基本情報を確認できるWebアプリケーションであり、主目的は自治体の公表している保育園空き状況を自動取得し、地図と紐付け、可視化することで引越し先の剪定や保育施設の建設計画の１資料として機能することである。

## 主な機能
- 地図上に保育園の位置をピン表示
- ホバー／クリックで保育園情報を表示
- 保育施設の空き状況の確認
- 詳細ページまたは外部HP（該当する保育園の施設情報）への遷移
